From User Flows to Attack Flows

"I use the same mental model for design and security—just with an adversarial lens."

Modelling Invariants

In design, an invariant might be "A user must be logged in to see this page." In security, we ask "Can a user force the system to show this page without being logged in?"

State Transitions

Most bugs happen in the 'space between' states. I map every possible transition and check for edge cases where the system might hang, de-sync, or allow an illegal move.
